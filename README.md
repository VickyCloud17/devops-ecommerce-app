# Cloud-Native E-Commerce DevOps Project

This is a production-style e-commerce platform built to demonstrate real-world DevOps practices.

## Architecture
- Microservices-based backend (Node.js)
- Frontend web UI
- Dockerized services
- Kubernetes orchestration
- GitOps with Argo CD
- AWS EC2 / EKS deployment

## Services
- user-service
- product-service
- order-service
- payment-service
